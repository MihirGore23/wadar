FrameStitchDemo

This is a simple program which demonstrates "frame stitching" on the X4; this program is based on the ClutterDemo project.

Hardware Setup

For this demo, a BeagleBone Black + Onza cape + LTSA 5 antennas were used. The orientation doesn't matter.

Demonstration Code

When a handle to the fex4 driver is opened, there is some immediate feedback; the X4 initialization performs several checks on startup. Typically, the INFO statements will happen quickly. On any failure, there are some retries which delay the output somewhat.

There is also some output shown about the X4 configuration.

Example - MTI3 Clutter Map

In this example, the radar was placed on a desk and the antennas were oriented horizontally. The MTI is very sensitive to motion, so any small movements near the radar show prominantly. The next large reflection was a coworker holding a metal plate in a hallway through an internal office wall. This is showing a frame which is ~20 m in length where the target is in the second frame.
