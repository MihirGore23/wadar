Detailed Description

Data structure used to represent an amplitude and index of value in array

Note
The idea here is that various peak search techniques will sort based on either the amplitude or index. Since we can't just sort an array directly by either like in matlab, we may construct an array of Peak_t then sort based on the chosen Peak_Search_e method.
