Detailed Description

The radar client is the complement to the radar server app which runs on the Beaglebone platform + radar cape.

The radarClient is a template for various specialized demonstration apps that use the radar server. It provides the basic client/server interaction. It uses the SalsaComm protocol as defined in the common demo folder.

Many of the other demo programs utilize Gnuplot with X11 forwarding for the radar signal visualization. In this scenario, Gnuplot is actually running on the Beaglebone platform. This limits the radar's framerate.

This provides basic client code which can be adapted for various demos. The header declares several functions which will be implemented for individual demo apps.
