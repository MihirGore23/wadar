ZoneSense

ZoneSense builds on the RangingDemo project by adding in some additional interpretation of the data.

Hardware Setup

For this demo, a BeagleBone Black + a Cayenne cape + Bowtie antenna cape was used. The optional antenna cape divider was also used; this addition made the signal look cleaner - the target reflections became more distinct.

Demonstration Code

This program builds on code from the RangingDemo project. In addition to the prior functionality, the program demonstrates the use of some DSP techniques.

In addition to the functionality present in the RangingDemo, this program adds some new features. It adds an envelope peak amplitude check which is used to determine if a target is in view. It also adds some boundary comparisons to the range estimates, where these boundaries define the "zones" that the target may be in.

Command-line Options

There are several command-line options:

  • -c : execute the radar timing measurement
  • -o : save the JSON configuration to a file
  • -a : enable the adaptive clutter map
