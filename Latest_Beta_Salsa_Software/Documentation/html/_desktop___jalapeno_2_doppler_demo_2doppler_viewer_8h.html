DopplerViewer

The Doppler Viewer is a simple API for visualizing data. It is intended to be used with the Salsa radar devices, so it is a tool to help view radar signals. It can be used on a desktop PC and should operate much faster than Gnuplot using piping.

Unlike the Salsa viewer, the Doppler viewer itself is used to perform DSP; the viewer contains the Doppler processing, the Doppler matrix, and some key analysis results. In addition to the update() and process() calls, there are a number of get() calls for analysis.

The DopplerViewer uses the SFML API for its graphics. SFML is a lightweight, multi-platform, multi-language API.

To install 'C' version of SFML on an Ubuntu system: 

sudo apt-get install libcsfml-dev

Make sure to add the following to the linking step in the makefile 

-lcsfml-graphics -lcsfml-window -lcsfml-system

Before using the viewer, it must be created 

DopplerViewer viewer;

// Create a viewer, 640 x 480 pixels where each signal is 512 samples, with a doppler length of 16 signals
viewer = DopplerViewer_create(640, 480, 512, 16, "../Common/resources/DejaVuSansMono.ttf");

The bottom portion of the window provides an area to display text. Text will display faster here than with stdout on the console. The user's app should provide storage for the string.

char userText[80];
...
// Update the string
snprintf(userText, 80, "Press [spacebar] to start");

// Update the text in the viewer window
DopplerViewer_updateText(viewer, userText);

On creation, the viewer is internally in a paused state. This has no effect on the use of most functions, but is provided just as a convenience for the user. Since this viewer runs on a desktop, a typical use might be to "play back" data collected previously for DSP development/experimentation.

Typical program flow: 

...
while (DopplerViewer_isOpen(viewer))
{
  // Get user input
  DopplerViewer_getInput(viewer);

  if (!DopplerViewer_isPaused(viewer)) {
    // code to get radar data
    // ...

    // Update the viewer and process
    DopplerViewer_updateData(viewer, radarSig);
    DopplerViewer_process(viewer);

    // code to analyze results
    // ...

    DopplerViewer_updateText(viewer, userText);
  }

  // Display the latest view
  DopplerViewer_display(viewer);
}
...

The DopplerViewer_getInput() provides support for several keys:

  • [ ESC ] - quits the program
  • [space] - toggles pause
