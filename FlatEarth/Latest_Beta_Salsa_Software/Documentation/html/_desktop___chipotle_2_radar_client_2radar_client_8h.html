Detailed Description

Declaration for common functions to be implemented by various demonstration apps which communication with the radar server.

The idea here is that the actual demonstration app would implement each of the functions defined here, which are not implemented in the radarClient.c file.

The getInput() is used to give additional keyboard command entries beyond the one's already supported by the SalsaViewer module.

The init() and free() functions are used to allocate the signal memory (and to free it) needed for the DSP.

The update() function is used to inject the latest radar counter values for each radar measurement.
