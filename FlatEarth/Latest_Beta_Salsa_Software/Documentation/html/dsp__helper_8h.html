void peakdetect_one ( const double *  x,
int  xn,
Peak_Search_e  method,
double *  peak_amp,
int *  peak_idx 
)

Function searches input signal for peaks and output the peak amplitude and index

This is an implementation of the peakdetect_v2() from matlab repo. This version is the equivalent of calling peakdetect_v2() with an argument of 'onemin', 'onemax', 'onefront', or 'onerear'. The output is then a single amplitude and its index.

Matches th matlab call: 

[peak_amp, peak_idx, ~] = peakdetect_v2(x, 'onemin');   % for min search
[peak_amp, peak_idx, ~] = peakdetect_v2(x, 'onemax');   % for max search
[peak_amp, peak_idx, ~] = peakdetect_v2(x, 'onefront'); % for front search
[peak_amp, peak_idx, ~] = peakdetect_v2(x, 'onerear');  % for rear search
Note
The matlab call peakdetect_v2() includes an option for pruning the signal. This implementation doesn't prune itself, but if that is desired then it can be accomplished with salsalib's radarDSP_dp_Prune() function.
If no peaks are found, then the peak_amp = 0, and peak_idx set to -1
If the user entered an invalid value for the search method, then it will default to a max search
Parameters
[in]*xInput signal
[in]*xnSignal length
[in]methodThe search method
[out]*peak_ampAmplitude of peak found via search method
[out]*peak_idxIndex into array of peak
