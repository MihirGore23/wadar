Function to send packet to remote system

Note
In this function, the <header><data> encoding happens internally. The code and length define the <header>. The <data> is the data buffer, which can be NULL if len == 0.
Parameters
[in]sockThe socket to send data out
[in]codeThe code which represents the packet's data context
[in]lenThe length of the <data> portion
[in]*dataThe data to send
