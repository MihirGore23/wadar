JalapenoComm

Declarations for a simple communications protocol

The Jalapeno Comm API encodes data into a <header><data> format.

The <header> is a 4 bytes, and encodes a uint16_t command code and a uint16_t data length. Thus, the maximal amount of data is 65535 bytes. In general, the length will be much smaller.

The <data> is a variable length array. The length is based on the type of data being sent.
