Detailed Description

BasicDemo

This is a simple program which demonstrates basic use of the fex4lib driver

Hardware Setup

For this demo, a BeagleBone Black + Onza cape + LTSA 5 antennas were used. The orientation doesn't matter.

Demonstration Code

When a handle to the fex4 driver is opened, there is some immediate feedback; the X4 initialization performs several checks on startup. Typically, the INFO statements will happen quickly. On any failure, there are some retries which delay the output somewhat.

Example - Basic Demo

For this example, the radar was placed on a desk and oriented to aim the antennas towards the ceiling. When the program runs, there is some output sent to the console.

Console output:

INFO: fex4: DVDD RX OK
INFO: fex4: AVDD TX OK
INFO: fex4: AVDD RX OK
INFO: fex4: XOSC lock: True

Raw data of ceiling

In this image, there are two large reflections. The first one is the direct-path reflection of the two antennas. The reflection of the ceiling is at sample 400.
